# KAIST-CS20200-Project
Term project repository for KAIST CS.20200 course

#### Project title: CLI 3-person Japanese Mahjong

This project aims to implement a single round of 3-person Japanese Mahjong in
command line. The user will be playing against two bots. The rules will follow
that from the *Mahjong Soul* online game platform.

#### Running the Project

Enter `dotnet run` to run the project. You will be greeted with the main menu,
where you can select your position and initiate the game. All interactions with
the game is done by entering a decimal number.
